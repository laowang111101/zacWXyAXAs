# Python考试考场管理系统 python953

## 项目概述

本项目是一个基于Python语言的考试考场管理系统，采用前后端分离的技术架构，旨在提供高效便捷的考试管理功能。

## 技术栈

- 后端：Python + Django
- 前端：Vue
- 数据库：MySQL

## 系统角色

- 管理员
- 工作人员
- 学生

系统采用前后端分离设计，前端使用Vue框架，后端使用Python和Django框架。若原电脑Node版本高于14，将提供远程配置cnpm环境服务。
